Hubungan HTML - CSS - Javascript

Kita terbiasa dengan tampilan website yang nampak pada browser, tetapi apakah kalian tahu bagaimana website dapat tampil demikian? Terdapat 3 (tiga) fondasi penting dalam membuat website. Yang pertama adalah HTML, yakni salah satu markup language, yang digunakan untuk membuat struktur dan menampilkan konten pada World Wide Web (Website). CSS dan JavaScript merupakan fondasi lainnya yang digunakan untuk mempercantik dan menjadikan website lebih dinamis dan interaktif. [dicoding.com]

Hubungan tersebut dapat digambarkan sebagai berikut.

Jika diibaratkan seperti bangunan, maka HTML merupakan pondasi semua aspek bangunan seperti tempat berdirinya tembok, pintu, atap, dan lain sebagainya. Sedangkan CSS sebagai perabotan hiasan yang mampu memperindah rumah sepeti hiasan rumah, wallpaper, karpet, dan lain sebagainya. Sedangkan Javascript berperan sebagai hal yang menghidupkan dan responsif seperti bisa untuk menyalakan lampu atau membukakan pintu.

Hypertext Markup Language (HTML)

"HTML adalah bahasa markup standar untuk halaman Web. Dengan HTML Anda dapat membuat situs web Anda sendiri. HTML mudah dipelajari - Anda akan menikmatinya!"[w3schools.com]

"Demikianlah tiga kalimat pembuka yang ada pada website w3schools.com terkait HTML. Dari ketiga kalimat tersebut, dapat kita ketahui bahwa, HTML merupakan bahasa markup standar untuk membuat halaman web. Untuk mempelajari sangat mudah, dan pasti kita akan menikmati pelajaran HTML ini, apalagi bagi yang memiliki motivasi kuat untuk bisa membuat aplikasi berbasis web.

HTML5 adalah versi HTML terbaru telah disempurnakan. Secara teknis, HTML bukan bahasa pemrograman, melainkan bahasa markup. Berikut ini materi yang akan dipelajari.

Struktur Dasar HTML Atribut HTML Heading Paragraf Link Images Tables List Form

Cascading Style Sheet (CSS)

"CSS adalah bahasa yang menggambarkan style dari dokumen HTML. CSS menjelaskan bagaimana elemen HTML harus ditampilkan."[w3schools.com]

Dari kutipan tersebut di atas, dapat dipahami bahwa CSS, akronim dari Cascading Style Sheet, digunakan untuk mengontrol setiap elemen yang ada pada dokumen HTML dengan cara yang sederhana dan mudah, sehingga tampilan dokumen HTML lebih indah dan menarik.

Menggunakan CSS adalah keharusan dalam mengembangkan aplikasi berbasis web, atau dengan kata lain jika ingin menjadi desainer web - mendesain HTML dan CSS adalah keterampilan yang harus dimiliki.

Adapun manfaat menggunakan CSS, sebagaimana disebutkan pada situs tutorialspoint.com antara lain:

  1.   Membuat situs web menakjubkan - CSS menangani tampilan dan nuansa bagian dari halaman web. Dengan CSS, Anda dapat mengontrol warna teks, jenis font, jarak antar paragraf, bagaimana ukuran kolom dan tata letak, gambar latar atau warna apa yang digunakan, desain layout, variasi tampilan untuk berbagai perangkat dan ukuran layar serta berbagai efek lainnya;
  2.   Kontrol web - CSS mudah dipelajari dan dipahami tetapi memberikan kontrol yang kuat atas penyajian dokumen HTML. Paling umum, CSS dikombinasikan dengan bahasa markup HTML atau XHTML.
  3.   Mempelajari bahasa lain - Setelah Anda memahami dasar HTML dan CSS, maka bahasa pemrograman untuk mengembangkan aplikasi berbasis web seperti javascript, php, atau angular menjadi lebih mudah dipahami.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang didukung oleh semua browser web modern. Javascript merupakan salah satu teknologi inti dari pengembangan aplikasi berbasis web, bersama dengan HTML dan CSS yang mungkin telah Anda pelajari sebelumnya.

Javascript merupakan salah satu bahasa pemrograman web yang bisa membuat halaman website Anda lebih menarik serta lebih hidup. Bahasa pemrograman javascript itu dijalankan di perangkat pengunjung website atau situs, bukan di server, sebagaimana bahasa pemrograman PHP.

Javascript membuat website menjadi lebih responsif, jadi jika tidak menggunakan javascript tidak masalah, tapi akan sangat berpengaruh pada tampilan website. Jika mengibaratkan javascript seperti rumah, mungkin tidak ada masalah jika tidak menggunakan lampu atau tidak dipasang pintu, tetapi hal ini tentu akan menimbulkan kesan rumah akan gelap, suram, dan kurang menarik. Begitu pula dengan website, tidak adanya javascript membuat website kurang menarik.
